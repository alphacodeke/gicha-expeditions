## 📋 Overview

**Gicha Nature Expeditions** is a modern, responsive tourism website designed for an eco-adventure company based in Nyeri, Kenya. The website showcases sustainable travel experiences, wildlife safaris, cultural tours, and nature expeditions while emphasizing environmental conservation and community-focused tourism.

## ✨ Features

- 🎨 **Modern & Responsive Design** - Fully responsive layout that works on all devices
- 🧭 **Interactive Navigation** - Smooth scrolling navigation with mobile hamburger menu
- 🎭 **Animated UI Elements** - Scroll reveal animations and interactive cards
- 📸 **Filterable Gallery** - Dynamic portfolio filtering by category (Wildlife, Cultural, Landscape)
- 📱 **WhatsApp Integration** - Contact form that sends messages directly via WhatsApp
- 🎯 **Call-to-Action Buttons** - Strategic CTAs for booking and inquiries
- 🌿 **Eco-Friendly Branding** - Nature-inspired design with sustainable tourism focus
